# 🏖️ 30초 해변 플로깅

국가해안쓰레기모니터링통계(MEIS) 데이터로 만든 30초 온라인 플로깅 챌린지.
게임 속에서 해안 쓰레기를 주우며 실제 해양 데이터를 소개하고, 데이터 활성화 성과를 정량 측정한다.

**▶ 지금 플레이: https://beach-plogging-flash-game.pages.dev**

---

## 게임 소개

30초 동안 화면 위로 나타나는 쓰레기를 손가락으로 끌어다 쓰레기통에 넣는 캐주얼 챌린지다. 단순한 미니게임이 아니라, 등장하는 쓰레기의 종류·확률·점수가 모두 실제 국가해안쓰레기 모니터링 데이터를 그대로 반영한다. 플레이하는 동안 자연스럽게 우리 바다의 쓰레기 실태(플라스틱 87.4% 등)를 접하게 되고, 그 데이터 접촉·인지·확산이 지표로 측정된다.

- **가입 없이 바로 참여** — 접속하면 해양생물 닉네임이 자동 생성되고 곧바로 플레이
- **데이터 기반 점수** — 실제로 드물게 발견되는 재질일수록 고득점(발생 확률의 역수)
- **떠내려오는 쓰레기 ×3, 콤보 ×2** — 빠르게 지나가는 쓰레기와 연속 수거에 보너스
- **보호 생물 감점** — 모래밭의 🦀 칠게 · 🪸 검붉은수지맨드라미 · 🪸 착생깃산호를 잘못 넣으면 −100점(바다 위를 헤엄치는 장수거북·물개 등은 배경이라 감점 없음)
- **쓰레기 도감** — 처음 잡은 재질이 9종 도감에 하나씩 채워짐
- **해안쓰레기 인사이트** — 게임 아래 실제 데이터 그래프(플라스틱 87.4% 등)
- **우리 바다 퀴즈** — 방금 본 데이터를 묻는 3문항(정답률·오답 분포까지 집계)
- **익명 랭킹** — 세션 기반, 최고 점수 자동 갱신, 닉네임·점수만 공개
- **인지 설문** — 데이터 인지 여부를 한 번의 탭으로 응답(학습 효과 측정)
- **바다친구 카드 공유** — 보호종이 고맙다고 인사하는 감사 카드 3종 중 하나가 랜덤으로 나와 카카오톡·인스타 등으로 공유

## 점수 등급

결과 화면에 표시된다. 공유 카드는 등급과 무관하게 3종 중 랜덤으로 나온다.

| 등급 | 기준 점수 |
|---|---|
| 🌟 바다지킴이 마스터 | 10,000점 이상 |
| 💪 열혈 플로거 | 6,000점 이상 |
| 🙂 새내기 플로거 | 1,000점 이상 |
| 🌱 플로깅 입문 | 1,000점 미만 |

## 성과 측정

이 프로젝트의 목적인 '데이터 활성화'를 다음 퍼널로 정량 측정한다.

```
방문 → 플레이 시작 → 완주 → (재플레이 · 랭킹 등록 · 데이터 클릭 · 공유 · 퀴즈 · 인지 응답)
```

핵심 지표는 **데이터 전환율**(플레이어 중 실제 MEIS 데이터를 클릭한 비율)로, 게임이 공공데이터 접근을 유도했는지를 보여준다. 그 외 인지 변화율(설문 '처음 알았어' 비율), 퀴즈 정답률·참여율, 공유율, 완주율을 발표용 대시보드(`stats.html`)에서 실시간 확인한다.

공유율과 퀴즈 참여율은 분모가 다르다. 결과 카드 공유는 결과 화면에만 있어 플레이어의 부분집합이지만, 하단 링크 공유와 퀴즈는 게임을 안 해도 닿을 수 있어 방문을 분모로 둔다 — 합쳐서 플레이어로 나누면 100%를 넘는다.

## 구조

```
public/          ← 게임 (Cloudflare Pages로 배포)
  index.html       메인 게임
  stats.html       발표용 지표 대시보드
  images/          공유용 감사 카드 이미지 3종(랜덤 노출)
api/             ← API (Cloudflare Workers로 배포)
  worker.js        랭킹 + 이벤트 수집 + 통계 집계
wrangler.toml    ← Worker 배포 설정 (D1 바인딩)
schema.sql       ← D1 테이블 스키마
```

## 기술 스택

- **프론트엔드** — 단일 HTML/CSS/JS (프레임워크·빌드 단계 없음)
- **백엔드** — Cloudflare Workers (서버리스 API)
- **데이터베이스** — Cloudflare D1 (SQLite)
- **호스팅** — Cloudflare Pages
- 전부 무료 티어로 운영, GitHub 연동 자동 배포

## 데이터 출처

국가해안쓰레기모니터링통계 (해양환경정보포털 MEIS)
https://www.meis.go.kr/mli/monitoringInfo/stat.do

2018~2023년, 전국 60개 조사지점. 게임 내 쓰레기 등장 확률·점수는 유형별 실측 개수(총 480,052개)를 그대로 반영한다. 대표 통계인 플라스틱 87.4%(419,456개)를 비롯한 9개 재질 비중이 데이터와 일치하며, 외부 출처나 추정치는 사용하지 않았다.
